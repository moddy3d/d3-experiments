tokens

documents being tokenized by a search engine
A document goes through several transformations when it is pushed into a search engine.

For illustration purposes, a piece of string: 
<\strong>A **Search Engine** is -SofTwArE- USED to search INFORMATION...<\/strong>
is pushed down the indexer where it is first stripped of HTML tags and symbols, then tokenized by whitespace. A lowercase filter is then applied to all the tokens.
